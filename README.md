# 🌾 Crop Production by District: Predictive Analytics for Agriculture


A comprehensive **Python-based Machine Learning project** developed for **WIA1006 Machine Learning Group Assignment** under the guidance of *Dr. Saw Shier Nee* and *Dr. Muhammad Shahreeza Safiruz bin Kassim*.
Created by **Group FemmeNet**.

---

## 📖 Overview
This project explores the **Crop Production by District** dataset to **predict whether crop production in a district is high or low** based on various agricultural features.
By applying multiple **classification algorithms**, the project aims to build models that can **accurately categorize production levels**, offering insights for better agricultural planning and resource allocation.

---

## ⚙️ Core Features
- ✅ Binary classification of production level (High/Low)
- 🔍 Handles missing values with Iterative Imputer  
- 🔁 Data preprocessing including feature encoding, scaling, and PCA 
- 🔗 Model evaluation using cross-validation and metrics  
- 💾 Implements multiple ML algorithms: Random Forest, Logistic Regression, XGBoost, LightGBM, CatBoost, Histogram-based Gradient Boosting  
- 📈 Hyperparameter tuning via RandomizedSearchCV
- 📊 Confusion matrices and feature importance visualization
- 📉 Outlier detection and handling
- 📅 Temporal feature extraction from date column

- **Libraries:** pandas, NumPy, scikit-learn, XGBoost, LightGBM, CatBoost, Matplotlib, Seaborn
